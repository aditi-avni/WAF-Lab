Definition

The View represents the user interface of an application. It displays the data received from the Model through the Controller and allows users to interact with the application.

Responsibilities

  • Display application data.
  • Provide an attractive user interface.
  • Accept user input.
  • Show messages and results.
  • Present information clearly.

Examples of User Interface Elements

  • Buttons
  • Text Boxes
  • Navigation Menus
  • Forms
  • Tables
  • Images
  • Drop-down Lists
